# Rolling Trunk Website Specification

## Project Name

Rolling Trunk Website

## Overview

This is a next.js website for The Rolling Trunk, a creative agency based in Milwaukee, Wisconsin. The website shows the logo located at ./rolling-trunk-logo.png. It uses three.js along with its ascii effect
at https://github.com/mrdoob/three.js/blob/master/examples/webgl_effects_ascii.html to create an ascii art effect. The website background should be a grid where there's a moving illuminated dot that is traversing the grid.

## Features

- Responsive design
- Mobile-first approach
- Clean and modern UI
- Smooth animations and transitions
- Interactive elements

## Technology Stack

- Next.js
- Tailwind CSS
- TypeScript
- React
- three.js
